# Jonnah Autonomous Dev System

This document defines the capabilities enabled for the Jonnah autonomous developer.

## Capabilities

### Error Log Analysis
Future logs from builds or runtime can be parsed to detect failures and improvement opportunities.

### Automatic Issue Planning
Potential improvements can be written into .jonnah/ISSUES.md and optionally converted to GitHub Issues.

### Feature Planning
Feature ideas are tracked in .jonnah/FEATURES.md.

### Code Generation
Planned areas for automated generation:
- React components
- API service layers
- basic CRUD endpoints

### Database Schema Planning
Future schema ideas will be stored in:

.jonnah/SCHEMA.md

### Development Loop
Every cycle:
1. Analyze repository
2. Generate improvement ideas
3. Apply small code updates
4. Commit and push

This system evolves the repository incrementally.
