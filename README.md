# Jonnah Workout

A Church Management System (CMS) dashboard designed to manage congregation data, church administration, and internal organizational structures for GKII communities.

## Overview

Jonnah Workout is a web-based administrative dashboard used to manage church data such as members, families, prayer groups, and church departments. It provides a modern interface for church administrators to organize and maintain congregation records efficiently.

This project originated from a dashboard template but is being evolved into a full church management platform.

## Core Features

- Congregation (Jemaat) management
- Family data management
- Prayer group (Kelompok Doa) management
- Church department/section management
- Administrative dashboard
- Data tables and charts
- Authentication UI

## Technology Stack

Frontend:
- React
- TypeScript
- Vite
- TailwindCSS

Backend:
- PHP API
- MySQL (expected database layer)

## Development

Install dependencies:

npm install

Run development server:

npm run dev

Build project:

npm run build
